What is the derivative of \( f(x) = x^2 \)?
A \( x \)
B \( 2x \)
C \( x^3 \)
D \( 2 \)

Correct!

The derivative of \( x^2 \) is \( 2x \) using the power rule.

Incorrect

The correct answer is B. Using the power rule: \( \frac{d}{dx}x^n = n x^{n-1} \), so \( \frac{d}{dx}x^2 = 2x^{2-1} = 2x \).

2
What is the derivative of \(\sin x\)?
A \(\cos x\)
B \( -\sin x \)
C \( -\cos x \)
D \(\tan x\)

Correct!

The derivative of \(\sin x\) is \(\cos x\).

Incorrect

The correct answer is A. The derivative of \(\sin x\) is \(\cos x\). This is a basic trigonometric derivative.

3
What is the value of \(\cos 0\)?
A 0
B 1
C \(-1\)
D Undefined

Correct!

\(\cos 0 = 1\). This is a fundamental trigonometric value.

Incorrect

The correct answer is B. \(\cos 0 = 1\). Remember that \(\cos\) of 0 degrees or 0 radians is 1.

4
What is meant by higher-order derivatives?
A Derivatives of only constants
B Repeated differentiation of a function
C Integration of a function
D Graph of a function

Correct!

Higher-order derivatives refer to taking derivatives repeatedly: first derivative, second derivative, third derivative, etc.

Incorrect

The correct answer is B. Higher-order derivatives are obtained by repeatedly differentiating a function. For example, the second derivative is the derivative of the first derivative.

5
Which of the following is a polynomial?
A \(\frac{1}{x}\)
B \(\sqrt{x}\)
C \(x^2 + 3x + 1\)
D \(\sin x\)

Correct!

\(x^2 + 3x + 1\) is a polynomial because it consists of terms with non-negative integer powers of x.

Incorrect

The correct answer is C. A polynomial is an expression consisting of variables and coefficients, involving only addition, subtraction, multiplication, and non-negative integer exponents. \(x^2 + 3x + 1\) fits this definition.

6
What is the derivative of \(e^x\)?
A \(xe^x\)
B \(e^x\)
C \(\ln x\)
D \(x^x\)

Correct!

The derivative of \(e^x\) is \(e^x\). This is a unique property of the exponential function.

Incorrect

The correct answer is B. The derivative of \(e^x\) is \(e^x\). This is a special property of the exponential function with base e.

7
Expand \((x + 1)^2\).
A \(x^2 + 1\)
B \(x^2 + 2x + 1\)
C \(x^2 + x + 1\)
D \(2x^2 + 1\)

Correct!

\((x + 1)^2 = x^2 + 2x + 1\) using the formula \((a+b)^2 = a^2 + 2ab + b^2\).

Incorrect

The correct answer is B. \((x + 1)^2 = (x+1)(x+1) = x^2 + x + x + 1 = x^2 + 2x + 1\).

8
What does approximation of a function mean?
A Finding exact value always
B Replacing a function with a simpler expression
C Differentiating a function
D Integrating a function

Correct!

Approximation means replacing a complex function with a simpler expression that is close to the original function.

Incorrect

The correct answer is B. Approximation of a function means replacing it with a simpler expression (like a polynomial) that is close to the original function, often to make calculations easier.

9
A function is continuous at a point if:
A It is differentiable there
B Its limit exists and equals the function value
C Its derivative is zero
D It is a polynomial

Correct!

A function is continuous at a point if the limit exists at that point and equals the function's value at that point.

Incorrect

The correct answer is B. A function f is continuous at a point c if: 1) f(c) is defined, 2) lim(x→c) f(x) exists, and 3) lim(x→c) f(x) = f(c).

10
Why are approximations used in mathematics?
A To avoid calculations
B To make problems more complex
C To simplify calculations and get near values
D To find exact answers only

Correct!

Approximations are used to simplify complex calculations and obtain values that are close enough to the exact answer for practical purposes.

Incorrect

The correct answer is C. Approximations are used to simplify calculations and get values that are close to the exact answer, especially when exact solutions are difficult or impossible to find.
