The Taylor series expansion of \( e^x \) about \( x = 0 \) is:
A \( 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \ldots \)
B \( 1 - x + \frac{x^2}{2!} - \frac{x^3}{3!} + \ldots \)
C \( x + \frac{x^2}{2} + \frac{x^3}{3} + \ldots \)
D \( 1 + x^2 + x^3 + \ldots \)

Correct!

\( e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!} \) is the standard Maclaurin series.

Incorrect

The correct expansion is \( 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \cdots \) (Option A).

2
The value of \( e^{0.5} \) using first 3 terms is approximately:
A 1.5
B 1.625
C 1.75
D 2

Correct!

\( e^{0.5} \approx 1 + 0.5 + \frac{(0.5)^2}{2} = 1 + 0.5 + 0.125 = 1.625 \) (Option B).

Incorrect

First 3 terms: \( 1 + 0.5 + 0.125 = 1.625 \).

3
The Taylor expansion of \( \sin x \) about \( x = 0 \) is:
A \( x - \frac{x^3}{3!} + \frac{x^5}{5!} - \ldots \)
B \( 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \ldots \)
C \( x + \frac{x^2}{2!} + \frac{x^3}{3!} \)
D \( 1 + x + x^2 \)

Correct!

\( \sin x = x - \frac{x^3}{6} + \frac{x^5}{120} - ...\) which matches option A.

Incorrect

\( \sin x \) expansion: \( x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots \) (Option A).

4
Approximate value of \( \sin (0.2) \) using Taylor expansion up to 2nd order is:
A 0.2
B 0.198
C 0.202
D 0.18

Correct!

\( \sin x \approx x \) (1st order) ⇒ 0.2 gives 0.2. Up to 2nd order (the cubic term is order 3), so still 0.2.

Incorrect

First non-zero term: \( x = 0.2 \) gives 0.2. Option A is correct.

5
The Taylor series of \( \ln (1 + x) \) about \( x = 0 \) is:
A \( x - \frac{x^2}{2} + \frac{x^3}{3} - \ldots \)
B \( 1 + x + \frac{x^2}{2} + \ldots \)
C \( x + x^2 + x^3 + \ldots \)
D \( 1 - x + x^2 - \cdots \)

Correct!

\( \ln(1+x) = \sum_{n=1}^{\infty} (-1)^{n+1} \frac{x^n}{n} = x - x^2/2 + x^3/3 - \cdots \)

Incorrect

Correct expansion: \( x - \frac{x^2}{2} + \frac{x^3}{3} - \cdots \) (Option A).

6
Approximate value of ln(1.1) using first 3 terms is:
A 0.095
B 0.105
C 0.1
D 0.11

Correct!

\( \ln(1.1) \approx 0.1 - \frac{0.01}{2} + \frac{0.001}{3} = 0.1 - 0.005 + 0.000333 ≈ 0.09533 \) → 0.095.

Incorrect

First 3 terms: \( x - x^2/2 + x^3/3 \) with x=0.1 gives ≈ 0.0953. Option A.

7
The Taylor expansion of \( \cos x \) about \( x = 0 \) is:
A \( 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots \)
B \( x - \frac{x^3}{3!} + \cdots \)
C \( 1 + x + x^2 \)
D \( x + x^2 + x^3 \)

Correct!

\( \cos x = 1 - \frac{x^2}{2} + \frac{x^4}{24} - \cdots \)

Incorrect

Cos series: \( 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots \) (Option A).

8
Approximate value of \( \cos (0.3) \) using Taylor expansion up to 2nd order is:
A 0.955
B 0.91
C 1
D 0.85

Correct!

\( \cos x \approx 1 - x^2/2 \) ⇒ \( 1 - 0.045 = 0.955 \) (Option A).

Incorrect

2nd order: \( 1 - (0.3)^2/2 = 1 - 0.045 = 0.955 \) → Option A.

9
The Taylor series of \( \frac{1}{1-x} \) about \( x = 0 \) is:
A \( 1 + x + x^2 + x^3 + \cdots \)
B \( 1 - x + x^2 - x^3 + \cdots \)
C \( x + x^2 + x^3 \)
D \( 1 + x^2 + x^3 \)

Correct!

\( \frac{1}{1-x} = \sum_{n=0}^{\infty} x^n = 1 + x + x^2 + ... \) for |x|<1.

Incorrect

The geometric series: \( 1 + x + x^2 + x^3 + \cdots \) (Option A).

10
The error in Taylor approximation generally decreases when:
A Degree of polynomial increases
B Degree decreases
C Value of x increases
D Function is constant

Correct!

Higher-degree Taylor polynomials typically reduce approximation error within the radius of convergence.

Incorrect

Error generally diminishes as the polynomial degree increases (Option A).
